What is DOM,whats the purpose of DOM.

The Document Object Model (DOM) is a web document programming interface. It represents the page so that programs can change the structure, style, and content of the document. which is represented by the DOM as nodes and objects. It also allow programming languages to interact with the page.

what is event booble.

Event Bubble is a DOM concept . It occurs when an element receives an action. And that event bubbles up through the DOM to its parent and grandparent elements, until it reaches the primary element.

What is callback function.

A callback function is a function that receives an input from another function and then call from inside the outer function to finish a task.
